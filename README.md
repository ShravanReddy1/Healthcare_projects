This repository showcases multiple healthcare data analytics and machine learning projects aimed at solving real-world healthcare challenges.

Projects Included
CHD Risk Prediction
Description: Developed machine learning models to predict 10-year coronary heart disease (CHD) risk using the Framingham Heart Study dataset.
Key Tools: Python (Scikit-learn), Logistic Regression, Random Forest, SVM.
Highlights: Explored class imbalance, key predictors, and model performance evaluation.
